About the oceans

The ocean is the body of salt water which covers approximately 71% of the surface of the Earth and contains 97% of Earth's water.It contains about 1.35 billion cubic kilometers of water.The ocean makes all life on Earth possible, and makes the planet appear blue when viewed from space. About Java,Python and PHP

Java is a general-purpose computer-programming language that is concurrent, class-based, object-orienated, and specifically designed to have as few implentation dependance as possible. In Java, programs are not compiled into executable files; they are compiled into bytecode(as disscused earlier), which the JVM(Java Virtual Machine)then executes to run time.

Python is an interpreted,high level,general purpose programming language.Created by Guido van Rossum and first released in 1991,Python has a design philosophy that emphasizes code readability,notably using significant whitespace.It provides,constucts that enable clear programming on both small and large scales.

PHP(arecursive acroyn for PHP: Hypertext preprocessor ) is a widely-used open source general -purpose scripting language that is especially suited for web development and can be embedded into HTML. It was orginally created by Rasmus lerdorf in 1994 ; the PHP reference implementation is now produced by the PHP group .
